# FunASR 语音识别测试报告

## 项目概述

最近用 FunASR 跑了一组语音识别测试，选的是 paraformer-zh 模型，在自己整理的数据集上评估了一下效果。测试数据是 500 条中文短语音，内容比较杂，有地名、机构名、数字、新闻播报这些。


## 测试环境

- **模型**：paraformer-zh
- **数据集**：ASHELL-2，放在data文件夹中
- **GPU**：CUDA
- **批处理大小**：16
- **VAD（语音活动检测）**：关
- **标点恢复**：关
- **测试样本数**：500
- **总耗时**：约 90 秒


## 测试结果

| 指标 | 结果 |
|------|------|
| **字错误率（CER）** | **2.80%** |
| **词错误率（WER）** | **21.40%** |



## 结果分析

**CER 2.80% 表现优异**，平均每 100 个字里大概错 2.8 个字，对于中文语音识别来说属于可用水平。日常场景下这个准确率已经能应付不少需求了。

**WER 21.40% 偏高**。看了一下结果文件，问题主要出在两类内容上：

一是专有名词，比如"闽宁镇""中国人民保险集团股份有限公司"这种长地名和机构名，模型识别时经常出错。二是数字表达，"四百七十六点九"这种读法，模型有时候会识别成 "476.9"，格式对不上导致 WER 被拉高。

**速度方面尚可**，500 条音频 90 秒跑完，平均一条不到 0.2 秒，实际用起来响应很快。


## 后续安排
FUNASR体量较大，后续若确定使用paraformer-zh模型，则将改模型从FUNASR中分离，并且进一步训练，以达到轻量化的效果
